# X's Hidden Lair

AI 训练师个人作品站。

## 快速开始

```bash
npm install
npm run dev
```

## 技术栈

- React
- TypeScript
- Vite
- React Router
- Three.js
- Framer Motion
- Tailwind CSS

## GitHub Pages 部署

1. 仓库默认分支使用 `main`
2. GitHub 仓库设置中启用 `Settings -> Pages -> Build and deployment -> GitHub Actions`
3. 推送到 `main` 后自动部署
4. 站点地址：
   - `https://mai74672691op-del.github.io/X.github.io/`
